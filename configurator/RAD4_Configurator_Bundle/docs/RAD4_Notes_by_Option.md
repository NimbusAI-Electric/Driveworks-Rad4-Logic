# RAD4 Notes Consolidated by CPN Option Code

For each option suffix in the CPN (excluding the 4-character finish codes such as `BR02` or `CH04`), this document lists the most common **Specification / Attention / Defogger / Dimming** notes that appear on the sales aids whose CPN includes that option.

Counts shown after each variant are the number of **CPNs with this option** that carry that particular note variant. For comparison, the parenthetical next to each option header (`N CPNs`) is the total number of CPNs that include the option.

Source: 381 RAD4 sales-aid PDFs in `D:\EM-HV-04 Backup 5-14-2026\Sales Aids\`.
